# SimpleCAD 2D

A small 2D CAD-style drawing app built with Python's standard Tkinter GUI toolkit.

## Run

```powershell
py app.py
```

## Features

- Draw lines, arrows, rectangles, circles, ovals, arcs, triangles, diamonds, and text labels
- Select and drag shapes
- Delete, clear, and undo changes
- Toggle snap-to-grid and grid visibility
- Pick stroke color and line width
- Save/load drawings as JSON
- Export the canvas as PostScript

## Shortcuts

- `Ctrl+S`: save
- `Ctrl+O`: load
- `Ctrl+Z`: undo
- `Delete`: delete selected shape
